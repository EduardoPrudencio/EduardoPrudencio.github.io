Programa criado para processamento e validação de dados do Bilhete Único

Este sistema foi desenvoilvido para a empresa responsável pelo procesamento de dados do Bilhete Único.

Suas principais funções são validar dados presentes no nome de cada arquivo, validar os dados nas linhas de cada arquivo, criar as tabelas no Banco de Dados e executar a inserção e atualização das informaçãoes nestas tabelas, tudo isso durante o processamento sem a necessidade de intervenção humana.

Mesmo com o grande número de linhas que um arquivo pode conter (mais de 1.000.000), o novo projeto reduziu o tempo de processamento para aproximadamente 1/4 do tempo que a antiga ferramenta levava.

O desenvolvimento foi feito em C# tentando aproveitar ao máximo alguns dos recursos que a linguagem oferece para paralelismo. A interface foi feita com uso de WPF(XAML).

Atuação no projeto: Sozinho
